# Program calculate matrix
## Addiction, Subtraction, Multiplication, Determinant
## Currently, Linux is only supported

### Prerequisites

Before running the program, make sure you have the following installed:

- **GNU Compiler Collection**
  ```bash
  gcc -v
  ```
- **Make** (GNU Make)
  ```bash
  make -v
  ```
  - Linux: usually pre-installed

---

### For compile
```
make

```
### For run
```
make run
```
### For clean
```
make clean
```
### For run test
```
make test
```
