# Project Matrix

A command-line matrix calculator written in **C**, supporting the four fundamental matrix operations: addition, subtraction, multiplication, and determinant calculation.

> This was built as a high school learning project. Feedback and bug reports are welcome!

---

## Features

| Operation      | Description                                      |
|----------------|--------------------------------------------------|
| Addition       | Add two matrices of the same dimensions          |
| Subtraction    | Subtract one matrix from another                 |
| Multiplication | Multiply two compatible matrices                 |
| Determinant    | Compute the determinant of a square matrix       |

---

## Requirements

- **GCC** – GNU C Compiler
- **Bash** – Unix shell (Linux / macOS / WSL on Windows)

To verify you have them installed:

```bash
gcc --version
bash --version
```

---

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/Loveberland/Project_Matrix.git
cd Project_Matrix
```

### 2. Run the program

```bash
bash start.sh
```

The `start.sh` script will compile the C source files and launch the program automatically.
